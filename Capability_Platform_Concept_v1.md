# Capability Platform Concept – Keep It Ultra Simple

**Core Principle**
Don't build another LMS. Build a lightweight workforce capability platform that delivers only what employees need to perform their jobs today. Every item must be immediately useful, current, and easy to consume.

## Design Philosophy

Instead of asking, 'What content should we store?', ask, 'What does an employee need to perform today?' Everything else stays out.

## The Four Core Experiences

### 1. Learn

Short HTML Capability Cards.
- One topic
- 2–10 minutes
- Interactive
- Mobile-first
- Ends with a knowledge check

Examples: New Hire Day 1, Battery Diagnostic, Customer Complaint Handling.

### 2. Verify

Simple confirmation that a skill was demonstrated.
- Supervisor sign-off
- Self-attestation where appropriate
- Builds a practical skills record instead of completion statistics.

### 3. Ask

AI assistant limited to approved company knowledge.
- Searches only trusted content
- Returns answer with source and last update
- No open-ended internet responses.

### 4. Updates

A single monthly operational update.
- New policy
- Procedure change
- Safety alert
- Product update

Employees review it in minutes and acknowledge completion.

## Monthly Subscription Deliverable

Sell one Capability Improvement each month rather than promising a fixed type of content.

A Capability Improvement could be:
- New HTML Capability Card
- Updated SOP
- AI knowledge refinement
- New checklist
- New verification workflow
- Process improvement

## Simple Home Screen

Today's Priorities

- Learn – Today's Capability Card
- Verify – Demonstrate today's skill
- Ask – Search approved company knowledge
- Updates – What's changed this month

## The Gold Standard Rule

Every item must pass all five tests:
1. Improves performance.
2. Will be used this month.
3. Can be completed in under 10 minutes.
4. Is current and maintained.
5. Would be missed if removed.

If any answer is 'No,' it does not belong.

## Long-Term Vision

Replace the traditional concept of 'courses' with Capability Cards. Employees receive only the knowledge, guidance, verification, or updates required to perform their work. This creates a focused operational readiness platform rather than a document repository.
